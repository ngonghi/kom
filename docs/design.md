# KOM

## 概要
Facebookの自分の投稿した画像をGoogle Driveに移動する。
（うちの子の可愛い写真をバックアップするため　www）

## 機能

- 基本機能
   - Facebookアカウントでログインできる
   - FBに投稿した画像をGoogle Driveに移動する
   - サイトにアクセスするため、username/passが必要だ
- 拡張機能
   - バックアップの状態をわかる
   - 画像のスライドショー

## サイトマップ
- /login: Facebookアカウントをログインするページ
- /albums: アルバム一覧ページ
	- /photos/{album_id}: アルバムの一覧画像ページ
- /slide_show: スライドショー
